# Contributing

This repo is a living resource. If you've built something real and it belongs here, add it.

## What to Add

- New n8n workflow templates (tested in production, not theory)
- Prompt library additions (real prompts you actually use)
- CRM integration guides for other platforms (Salesforce, Pipedrive, Attio)
- Email provider alternatives (SendGrid, Postmark, Mailgun)
- Raycast extension patterns and templates
- Corrections and improvements to existing guides

## What Not to Add

- Untested theory
- Tool advertisements or affiliate links
- API keys, credentials, or client-specific data
- Copied content from other sources

## How

1. Fork the repo
2. Add or edit in the relevant track folder
3. Open a PR with a one-line description of what you're adding and why

Keep it practical. If it can be said in 3 lines, don't write 20.
