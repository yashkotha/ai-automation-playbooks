# AI Automation Playbooks

Real systems, built in production. Not demos or tutorials. Actual patterns, templates, and guides from building AI-first workflows across marketing, CRM, design, and developer tooling.

This repo covers 4 communities:

- **AI builders** - n8n, Claude Code, self-hosted stacks, prompt engineering
- **Marketers** - AI email sequences, lead scoring, multi-touch automation
- **HubSpot users** - CRM architecture, custom properties, n8n integrations
- **Designers and devs** - Framer + AI, design-to-code, Raycast extensions

Everything here has been run in production. Use it, build on it, contribute back.

## The Stack

| Layer | Tool | Cost |
|-------|------|------|
| Automation | n8n (self-hosted) | Free |
| Compute | VPS (Hetzner / Hostinger / DigitalOcean) | ~$7/month |
| AI | OpenAI API | ~$10-20/month |
| Email | Resend | Free up to 3k/month |
| CRM | HubSpot (free tier) | Free |

Under $30/month total. No SaaS sprawl.

---

## Philosophy

Most AI content online is either too abstract or too tied to one vendor. The patterns here work regardless of which tools you pick. Swap n8n for Make, Resend for SendGrid, HubSpot for Pipedrive. The architecture stays the same.

Start with whichever track matters most to you. Each folder has a README that tells you where to begin.
